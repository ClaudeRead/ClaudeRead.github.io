## ⚙️ 说明

&emsp;&emsp;最近用AI大模型尝试开发了一个讲义生成系统，这个系统尝试解决如下这个情况：
- 本地有一个知识库，如果现在去做一个培训，恰好培训的内容是你在知识库当中平时记录的内容，现在需要将知识库中记录的笔记转化成讲义。一般情况下，需要把每个笔记翻开一个个看，然后根据面向不同的人群，在把内容一个个复制到Word。这样效率十分低下，那么现在有了这个系统，只需维护知识库，就可以在Web界面上根据大纲选取内容，你可以控制是否显示在最后的生成的讲义当中带有“【】”中的内容(有可能是你的小记、或者注意之类的提示内容)，最后生成属于自己的讲义。

## ⚙️ 技术栈

- **后端**：Python 3.6+，Flask框架
- **前端**：HTML5，CSS3，JavaScript
- **依赖**：
   - Flask：Web框架
   - Flask-CORS：跨域支持
   - markdown：Markdown解析

## 🚀 快速开始

### 方法二：手动安装

1. **拉取项目**
   ```bash
   git clone https://github.com/ClaudeRead/ObsidianProject.git
   ```

2. **进入目录**
  ```bash
   cd ObsidianProject
   ```

3. **安装依赖**
   ```bash
   pip install -r lecture_generator/requirements.txt
   ```

5. **启动应用**
   ```bash
   python run.py
   ```

5. **访问界面**
   打开浏览器访问：http://localhost:5000

## 📋 使用流程

1. **设置路径**：选择知识库所存放的路径
2. **选择文件**：在左侧文件树中勾选需要的文件
3. **配置选项**：设置是否显示标签内容，选择输出格式
4. **预览内容**：点击"预览讲义"查看效果
5. **生成下载**：点击"生成讲义"并下载文件
