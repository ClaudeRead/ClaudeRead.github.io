## Network Communication
- What protocol maps IP addresses to MAC addresses?(哪种协议将 IP 地址映射到 MAC 地址？)
 > ARP

- Which IP version uses 128-bit addressing?(哪种 IP 版本使用 128 位寻址？)
 > IPV6

- At which layer of the OSI model do ports operate? (Format: two words)(端口号在 OSI 模型的哪一层运行？（格式：两个词）)
 > Transport Layer

- What is the designated port number for HTTP?(HTTP 的指定端口号是多少？)
 > 80

- What is the first step in the process of a web browsing session? (Format: two words)(浏览器会话的第一步是什么？（格式：两个词）)
 > DNS Lookup

## Dynamic Host Configuration Protocol

- What protocol automates IP address configuration for devices on a network?(哪种协议自动配置网络上的设备 IP 地址？)
 > DHCP

- What acronym describes the sequence of messages exchanged during the DHCP process?（哪个缩写词描述了 DHCP 过程中交换的消息序列？）
 > DORA

- What type of message does a client send to accept an IP address from a DHCP server?(客户端发送哪种类型的信息以接受来自 DHCP 服务器的 IP 地址？)
 > REQUEST

## Network Address Translation (NAT)
- What type of NAT allows multiple private IP addresses to share one public IP address using unique port numbers?
 > PAT

- What RFC specifies private IP ranges?
 > RFC 1918

- Which NAT type involves a one-to-one mapping of private IP addresses to public IP addresses?(哪种 NAT 类型涉及私有 IP 地址与公共 IP 地址的一对一映射？)
 > Static NAT

- What type of NAT assigns a public IP from a pool as needed?(哪种 NAT 会根据需要从池中分配公共 IP 地址？)
 > Dynamic NAT

- What device typically performs NAT in a home network?(哪种设备通常在家网络中执行 NAT？)
 > Router

## Domain Name System (DNS)
- What type of domain is `.com` considered as? (Format: Three words, example: One-Two Three)(`.com` 被视为哪种类型的域名？（格式：三个词，例如：一级 二级 三级）)
 > Top-Level Domain

- In the domain `www.example.com`, what is `example` called?(在域名 `www.example.com` 中，`example` 被称为什么？
)
 > Second-Level Domain

- What is checked first in the DNS resolution process when you enter a domain name into a browser? (Format: Two words)(当你将域名输入浏览器时，DNS 解析过程中最先被检查的是什么？(格式：两个词))
 > DNS cache

- What type of DNS server is typically provided by an Internet Service Provider?(互联网服务提供商通常提供哪种类型的 DNS 服务器？)
 > ecursive DNS server

- Which server directs the recursive DNS server to the appropriate TLD name server?(哪个服务器指导递归 DNS 服务器到正确的顶级域名服务器？)
 > root server

- What numerical label uniquely identifies a device on a network?(什么数字标签唯一标识网络上的一个设备？)
 > IP Address

- In the URL "accounts.google.com", what is `accounts` considered as?(在 URL "accounts.google.com"中，`accounts`被视为什么？)
 > Subdomain
